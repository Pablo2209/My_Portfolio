Fresenius Kabi

Project for supervision and control of new production lines.

My main functions include the programming of the control system, using a Siemens S7-400 controller.

The main control network included an ASi-Bus network and a Profibus network was used for the communication of sensors and measurements of variables.

Currently the system is working and producing medicine which is packed in glass ampoules.

Autopista Central

Control system, whose purpose is to extract rainwater from the low-level crossings that are found along the Autopista Central.

Project which I lead, developing all the control logic for the PLCs, in addition to extracting data through ModBus communication, the control of drives and supervision of sensors installed in the technical booths.

Transelec

Design and implementation of electrical variables measurement system for high voltage towers located in the north of Chile. Among which are; measurement of temperature, inclination and voltage of the conductor.

In addition, a system for measuring meteorological variables is implemented; Ambient temperature, relative humidity, rain, etc.

These equipments help to determine possible electrical discharges that can occur mainly due to the humidity of the environment and the dirt of the cables.
